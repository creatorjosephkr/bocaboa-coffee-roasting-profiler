GitHub 연동 설정

GitHub 개인 저장소 및 PAT 생성 안내

1. GitHub 계정 만들기 및 로그인

GitHub 계정이 없다면 github.com에서 무료로 가입 후 로그인해 주세요.

2. 새 저장소(Repository) 만들기

로그인 후 우측 상단의 [+] ➔ New repository를 클릭합니다.

  • Repository name: 원하는 이름 입력 (예: bocaboa-profile)
  • Public / Private: 비공개 보관을 원하시면 Private를 선택합니다.
  • 맨 아래 Create repository 버튼을 누르면 빈 저장소가 생성됩니다.

3. 개인 토큰(PAT) 발급하기

우측 상단 프로필 이미지 클릭 ➔ Settings 선택 ➔ 왼쪽 맨 아래 Developer settings 선택 ➔ Personal access tokens ➔ Tokens (classic) 선택.

  • 우측 상단 Generate new token ➔ Generate new token (classic) 클릭.
  • Note: 토큰 이름 입력 (예: BocaBoa)
  • Expiration: 만료일 선택 (예: No expiration 또는 90 days)
  • Select scopes: 저장소 읽기/쓰기를 위해 맨 위의 [ ] repo 체크박스에 체크합니다.
  • 맨 아래 Generate token을 클릭하면 토큰 값(ghp_...)이 표시됩니다. 이 값을 복사하여 앱에 입력합니다. (창을 닫으면 다시 볼 수 없습니다.)
